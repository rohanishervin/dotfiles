# Dotfiles

## Dependencies:
 - neovim
 - kitty
 - font awesome 
 - powerline fonts
 - powerleve9k
 - bumblebee-status
 - i3-style

## Installation
clone the repository, 
```bash
git clone --recurse-submodules https://github.com/rohanishervin/dotfiles.git ~/.dotfiles
```


then:

```bash
cd .dotfiles
bash install.sh
```
